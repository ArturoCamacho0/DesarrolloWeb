El Scope es el alcance que tiene una variable o función, ya sea global o local

El Scope global funciona en todo el script

El Scope local solo funciona dentro de dónde se está ejecutando(función)

El callstack es una pila de ejecución que guarda y saca los pasos que debe realizar el navegador, el último en entrar es el primero en salir.

Anonymus es la primera y última ejecución a la hora de ejecutar, cuando sale del callstack es que ya terminó la ejecución del script.
